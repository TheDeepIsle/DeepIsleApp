# DeepIsleApp

## Download
Grab the latest installer from the [Releases page](https://github.com/TheDeepIsle/DeepIsleApp/releases/latest) — download `DeepIsleAppSetup.exe` and run it. The app will auto-update itself on future releases, so you only need to do this once.

### Requirements
- Windows 10/11
- An active connection to The Deep Isle or The Earl (no manual server setup needed — just log in)

## Features
- **Live interactive map** — real-time player positions on the Gateway map with pan/zoom
- **Proximity voice chat** — push-to-talk with fully customizable key/button binding (any key or mouse button)
- **Dino storage** — store and redeem your dino with all stats preserved
- **Admin tools** *(staff only)* — player management, activity logs, prime granting, and more

## Setting Up Push-to-Talk
1. Open the Voice panel
2. Click **Set PTT Button**
3. Press any key or mouse button you'd like to use — it'll be captured and labeled automatically
